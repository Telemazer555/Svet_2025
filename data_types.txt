int (целые числа) — представляют собой целочисленные значения. Например, x = 10. После создания переменной x изменить её значение без пересоздания нельзя.

float (числа с плавающей запятой) — числа с дробной частью, например, y = 3.14. Как и целые числа, они неизменяемы.

bool (логический тип) — принимает два значения: True и False. Например, flag = True. Логическое значение нельзя изменить после его присвоения.

str (строки) — последовательности символов. Например, text = "Hello". Нельзя изменить отдельные символы строки, но можно создать новую строку на основе существующей.

tuple (кортежи) — упорядоченные коллекции данных, аналогичные спискам, но неизменяемые. Например, t = (1, 2, 3). После создания кортежа его элементы нельзя изменить.

frozenset (замороженные множества) — неизменяемый аналог множества set. Например, fs = frozenset({1, 2, 3}). Элементы множества после создания не могут быть добавлены или удалены.
